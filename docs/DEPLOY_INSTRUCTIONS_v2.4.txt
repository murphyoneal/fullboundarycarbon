HOW TO DEPLOY (GitHub Desktop)
------------------------------
Repo at: C:\Users\<you>\Documents\GitHub\fullboundarycarbon\

  Step 1. Extract this package on your computer.

  Step 2. Copy the contents into your local repo folder, allowing
          overwrites:
            - revision-history.html  -> repo root (new)
            - index.html, codex.html, companions.html, about.html,
              methodology.html, feedback.html, legal.html,
              resources.html, 404.html, sitemap.xml -> repo root
              (overwrite)
            - docs\*.pdf -> repo \docs\ (4 files; Codex overwrites
              nothing since the filename is new — v4.3.3)

  Step 3. (Recommended) Delete \docs\DRL-Codex-v4.3.0.pdf from the
          repo so only the current Codex version is served.

  Step 4. Open GitHub Desktop. Confirm the change list:
          ~10 modified HTML/XML files, 1 new HTML page, 4 new
          PDFs, 1 deleted PDF (if you did Step 3).

  Step 5. Summary box:
            "Site v2.4: Codex v4.3.3; Curtain, Appendix Z,
             Tree of Many Sales, ESG release; revision-history page"
          Commit to main, then Push origin.

  Step 6. Cloudflare auto-deploys within ~60 seconds.
