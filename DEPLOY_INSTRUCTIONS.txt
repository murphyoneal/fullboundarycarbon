HOW TO DEPLOY (GitHub Desktop, one-shot)
----------------------------------------
You already have GitHub Desktop set up and `murphyoneal/fullboundarycarbon`
cloned locally to:
    C:\Users\<you>\Documents\GitHub\fullboundarycarbon\

The cleanest sequence:

  Step 1. Open File Explorer to your local repo folder.

  Step 2. Delete (or move out) everything currently in that folder
          EXCEPT the .git folder (which Windows usually hides — if
          you don't see it, that's fine, just leave it alone).
          The old index.html and any other v1 files need to be gone
          since v2.1 replaces them.

  Step 3. Extract site-v2.zip on your computer. Open the extracted
          site-v2 folder so you can see the contents: assets/, docs/,
          pages/, corrected-ledger/, reproduction-sheets/, and the
          loose .html files.

  Step 4. Select ALL the contents of the extracted site-v2 folder
          (Ctrl+A), copy (Ctrl+C), paste (Ctrl+V) into your GitHub
          repo local folder.

  Step 5. Open GitHub Desktop. It will show many changes — additions,
          modifications, possibly some deletions of v1 files. That's
          correct.

  Step 6. In the Summary box at the bottom type:
              "Site v2.1: full content, all docs, photo hero"
          Click "Commit to main".

  Step 7. Click "Push origin" at the top.

  Step 8. Cloudflare auto-deploys within ~60 seconds. Test at:
              https://fullboundarycarbon.pages.dev
